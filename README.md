Build Number Scheme:
Digits 4&5 - Build
Digit 3 - Subversion (1 means subversion 2, 2 means subversion 3, etc.)
Digits 1&2 - Version (10, 5, 21, etc.)

00104 means Version 0 Subversion 2 Build 04 or v0.2.
